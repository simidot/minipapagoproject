# minipapagoproject

#  🍳Overview

> 🧑‍🎓📚파파고와 비슷한 통역 사이트 기능을 구현한 사이트입니다.

자바스크립트의 기초와 네이버 통역 API 활용하여 통역 사이트 구현하였습니다.

## 📍 주요 기능
1. **입력 시 자동 언어 감지:**
    - 관리자는 전체 회원 목록을 조회하고 관리할 수 있다.

2. **번역 언어 변경:**
    - 관리자는 전체 회원 목록을 조회하고 관리할 수 있다.

3. **다크 모드 설정:**
    - 관리자는 새로운 멤버를 등록할 수 있다

      **_(프로필 이미지를 업로드하여 서버에 저장하고 해당 이미지를 멤버 프로필에 표시 가능)_**
